ChatGPT API Documentation

/api/chatgpt

Method: POST

Gets an answer from ChatGPT based on input data.

Request:
POST /api/chatgpt
Body: 
{
  "question": "Hello",
}

Response:
{
    "answer": "Hello, how can I assist you?",
    "model": "gpt-3.5-turbo",
    "provider": "OpenAI"
}
